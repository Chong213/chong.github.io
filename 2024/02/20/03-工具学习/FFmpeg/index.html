FFmpeg

https://ffmpeg.org/ffmpeg.html

ffmpeg是一个通用媒体转换器。

它可以读取各种输入 - 包括实时抓取/记录设备 - 过滤并将其转码为多种输出格式。

https://www.xjx100.cn/news/582790.html

1、使用语法：

  • 参数选项：
  • -an：去掉音频（audio no）
  • -acodec ：音频选项， 一般后面加copy表示拷贝
  • -vcodec：视频选项，一般后面加copy表示拷贝
  • 格式：
    • h264：表示输出的是h264的视频裸流
    • mp4：表示输出的是mp4的视频
    • mpegts：表示ts视频流s

2、功能

  1. 视频拼接

裸码流的拼接，先拼接裸码流，再做容器的封装

  1. 截取图片 -s：尺寸
  1. 转GIF -vframes：30帧
  1. 截取指定时间的缩微图，-ss后跟的时间单位为秒
  1. 视频->图片

3、部分指令

参数 解释
-y 覆盖输出文件
-fromats 显示可用的格式，编解码的，协议的。。。
-f fmt 强迫采用格式fmt（format）
-t duration 设置纪录时间 hh:mm:ss[.xxx]格式的记录时间也支持（time）
-ss position 搜索到指定的时间 [-]hh:mm:ss[.xxx]的格式也支持
-target type 设置目标文件类型(vcd,svcd,dvd) 所有的格式选项（比特率，编解码以及缓冲区大小）自动设置。
只需要输入如下的就可以了：ffmpeg -i myfile.avi -target vcd /tmp/vcd.mpg
-r fps 设置帧频 缺省25（rate）
-s size 设置帧（1帧就是一幅图像）大小 格式为WXH 缺省160X128。
-b bitrate 设置比特率，缺省200kb/s
-aspect aspect 设置横纵比 4:3 16:9 或 1.3333 1.7777（aspect：外观）
-vcodec code 强制使用codec编解码方式。如果用copy表示原始编解码数据必须被拷贝。
