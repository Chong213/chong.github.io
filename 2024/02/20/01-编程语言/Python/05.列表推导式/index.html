列表推导式

列表推导式是 Python 语言特有的一种语法结构，也可以看成是 Python 中一种独特的数据处理方式

**列表推导式会返回一个列表**，因此它适用于所有需要列表的场景。

例：

for循环

两层for循环

列表推导式也支持嵌套

字典推导式

元组推导式 && 集合推导式

使用元组推导式生成的结果并不是一个元组，而是一个生成器对象，需要特别注意下，这种写法在有的地方会把它叫做生成器语法，不叫做元组推导式。
