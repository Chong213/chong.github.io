GPU参数

1、GPU

简介

图形处理器（英语：graphics processing unit，缩写：GPU），又称显示核心、视觉处理器、显示芯片，是一种专门做图像和图形相关运算工作的微处理器。

图形的特点是每个像素处理所需的计算比较单一，但包含的像素点数量巨大，且对计算速度的要求很高。

2、参数

像素填充率和纹理填充率反映的是GPU的性能，而显存带宽则体现了显存的性能。

2.1、显存

  • 显存，全称显示内存，暂时储存显示芯片要处理的数据和处理完毕的数据。
  • 图形核心的性能愈强，需要的显存也就越多。
  • GPU拥有自己的内存，称为视频RAM（VRAM）或图形双数据率（GDDR）内存。

2.2、显存位宽

  • 显存位宽，指一个时钟周期内能传输数据的位数（bit）。
  • 显存位宽位数越大则瞬间所能传输的数据量越大

2.3、显存带宽

  • 显存带宽是指显示芯片与显存之间的数据传输速率，它以字节/秒为单位。
  • 显存带宽是决定显卡性能和速度最重要的因素之一
  • 显存带宽＝显存频率×显存位宽/8

2.4、像素填充速率

  • 像素填充速率，指GPU一秒钟内能处理多少个像素
  • 单位是GPixel/S（每秒十亿像素），或MPixel/S（每秒百万像素）。
  • 像素填充速率是较好衡量GPU图像显示功能的整体指标，说明了显卡能以多快的速度对图像进行光栅化处理

2.5、纹理填充率

  • 纹理填充率，指对多边形图像进行纹理贴图、实现3D效果的速度，和像素填充率类似
  • 单位是GTexels/S或MTexels/S

2.6、浮点性能

  • 浮点性能：代表显卡的浮点计算能力，越高算力越强，深度学习、科学计算用途较多
  • 理论峰值 ＝ GPU芯片数量＊GPU Boost主频＊核心数量＊单个时钟周期内能处理的浮点计算次数
  • $$PeakFLOPS = F_{clk}*N_{SM}*F_{req}$$
    • $$F_{clk} $$为GPU的时钟周期内指令执行数（单位为FLOPS/Cycle）
    • $$F_{req}$$为运行频率（单位为GHz）
    • $$N _{SM}$$为GPU SM数量（单位为Cores）

2.7、Core

CUDA Core
  • 可以执行 32 位浮点加/乘法、32 位到 8 位整数运算（如移位、加法、muls和类似操作）的管道（pipe）
  • 可以在其 SM 单元上生成内存请求以提供数据以不断处理数据，还要求特殊函数计算并由 SM 单元同步，以正确计算并行算法。
Tensor Core
  • 本质上，Tensor 核心是加速矩阵乘法过程的处理单元。
  • Tensor Core 支持混合精度计算，动态调整计算以加速吞吐量，同时保持准确性。
  • Tensor Core 的计算速度比 CUDA Core快得多。CUDA Core每个时钟周期执行一次操作，而Tensor Core每个时钟周期可以执行多个操作。
RT Core
  • RT 内核，工作是尽快完成光线追踪的数学计算。足够快，能够以可播放的帧速率在屏幕上显示移动图像
  • 简而言之，RT 核心向更通用的 CUDA 核心添加了额外的电路，当光线追踪计算出现时，这些电路可以包含在渲染管道中。CUDA 核心将这项工作交给 RT 核心，然后使用光线追踪数学的结果来渲染场景并正确地对眼球前面的像素进行着色。
